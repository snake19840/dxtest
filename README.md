# 电信设备管理系统

#### 介绍
机房管理,合同管理

#### 软件架构
软件架构说明:
前端:jsp ,vue , element, echarts,fullcalendar,JavaScript
后端:servlet ,java
数据库:oracle


#### 安装教程

1.  修改数据库连接帐号与密码
3.  建orcale数据库,文件位置/sql/TEST_01.sql;
4.  war包放入tomcat

#### 使用说明

1.  DXTESTUSER表中创建用户,权限级别都设置为1
2.  test帐号密码test
3.  登陆使用
